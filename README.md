# IoT Device to Count People (Open Hardware)
Hardware designed to count number of people based on number of mobile phones.

<h4>Technologies:</h4>
<ul>
  <li>ESP8166 - Espressif</li> 
  <li>Arduino IDE</li>
  <li>C/C++</li>
  <li>PCB designed on Eagle</li>
</ul>
